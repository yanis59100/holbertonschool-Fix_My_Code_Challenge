# Holberton School Fix My Code Challenge

This repository contains the "Fix My Code" challenge projects I completed during my studies at Holberton School.  
The focus of these projects was to practice debugging, code review, and problem-solving skills in various programming languages.

## Projects Overview
- Debugging and fixing broken code  
- Understanding common coding errors and pitfalls  
- Practicing problem-solving techniques  
- Reviewing and improving code quality  

## Tech / Stack
- Python  
- C  
- Bash / Shell scripting  
- Git for version control  
- Linux environment  

## What I Learned
- Identifying and fixing syntax and logical errors  
- Improving code readability and maintainability  
- Understanding best practices in coding  
- Enhancing debugging skills across multiple languages  

## Usage
Clone the repository and explore the challenge folders to see broken code, fixed solutions, and explanations.
